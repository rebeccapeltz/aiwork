## Security Best Practices

### Input Safety
- **Never** enter private information about yourself, family, friends, or workplace
- Avoid sharing sensitive data like passwords, financial info, or personal identifiers

### Output Awareness
- **Hallucinations** = AI errors or mistakes
- Always verify important information from reliable sources
- AI can confidently provide incorrect information

## Problem-Solving Approaches

### Chain of Thought (COT)
- Step-by-step process
- Shows reasoning
- Best for: Planning events, calculations, sequential tasks

### Tree of Thought (TOT)
- Multiple reasoning paths
- Explores different strategies
- Best for: Marketing strategies, complex decisions, multi-option scenarios

## Creating an Agent

Structure your prompt to create a specialized AI assistant:
```
I've uploaded [document description].
It contains [key information].

You are a [Role] who can [specific capabilities].
Answer questions about [topic area].
```

**Example:** Customer Service Agent, Teaching Assistant, Data Analyst

## Tips for Better Results

1. **Be Specific** - Provide detailed context and clear expectations
2. **Use Examples** - Show what you want with samples
3. **Iterate** - Refine prompts based on results
4. **Combine Modes** - Upload images, documents, and text together
5. **Set Constraints** - Specify length, format, tone, and style
6. **Ask for Reasoning** - Request step-by-step explanations
7. **Verify Critical Info** - Double-check important facts and figures

## Multimodal Prompting

Combine different input types for better results:
- Text + Images
- Text + Documents
- Text + Audio
- Multiple documents together

Upload files to provide context and reference material for more accurate responses.
