Key Concepts

Click each concept to expand. Use these as discussion prompts with your class.

1
What is a chatbot?
▾
A chatbot (like Claude.ai or ChatGPT) is a complete application. It bundles a user interface, conversation memory, safety filters, and access to an AI model — all in one package. You type, it responds. You don't see what's behind the curtain.

Think of it like a vending machine: press a button, get a snack. No need to know how the machine works.
Discussion: What are the limitations of only having the "vending machine" version of AI?
2
What is an API?
▾
An API (Application Programming Interface) is a way for software to talk to other software. When you use the AI API, you're sending a structured message directly to the model service — and getting a structured message back.

You control: the system prompt, the user message, the model, the temperature, and max_tokens. Nothing is hidden.

Think of it like talking directly to the chef instead of ordering from a menu.
Try it: Edit the system prompt and see how the model's personality changes completely.
3
What is a System Prompt?
▾
The system prompt is a set of instructions given to the model before the user says anything. In a commercial chatbot, this is set by the company and kept secret. When you use the API, you write it yourself.

System prompts can define: a persona, rules to follow, a tone of voice, topics to avoid, or even a character in a game.

"You are a sarcastic pirate who only speaks in rhyme..." — totally valid!
Key insight: Every AI chatbot you've ever used had a system prompt. You just couldn't see it.
4
Temperature & Tokens — the knobs
▾
Temperature controls how "creative" the model is. At 0, it picks the most likely next word (predictable, focused). At 1, it takes more risks (creative, surprising).

Tokens are the basic units the model processes — roughly ¾ of a word. max_tokens limits response length. You pay per token in production APIs.
Business context: Temperature 0 is best for data extraction. Temperature 0.7–1 is better for marketing copy and creative writing.
5
What is an AI Agent?
▾
An agent is when an AI model's output drives further actions — not just displays text. The model becomes a decision-maker in a workflow.

Example: User asks "weather in Paris?" → Model extracts "Paris" → Code calls a weather source → Data fed back to model → Model gives a full answer.

The model isn't browsing the internet itself. It's being orchestrated by code that calls tools on its behalf.
The key mental model: Agent = API call + decision logic + more API calls.
