Key Concepts

Click each concept to expand. Use these as discussion prompts with your class.

1
What is a chatbot?
▾
A chatbot (like Claude.ai or ChatGPT) is a complete application. It bundles together a user interface, conversation memory, safety filters, and access to an AI model — all in one package. You type, it responds. You don't see what's happening behind the curtain.

Think of it like a vending machine: you press a button and get a snack. You don't need to know how the machine works.
Discussion: What are the limitations of only being able to use the "vending machine" version of AI?
2
What is an API?
▾
An API (Application Programming Interface) is a way for software to talk to other software. When you use the AI API, you're sending a structured message directly to the model service — and getting a structured message back.

You control: the system prompt, the user message, the model, the temperature, and the max_tokens. Nothing is hidden.

Think of it like talking directly to the chef instead of ordering from a menu.
Try it: Edit the system prompt in the demo and see how the model's personality changes completely.
3
What is a System Prompt?
▾
The system prompt is a set of instructions given to the model before the user says anything. In a commercial chatbot, this is set by the company and kept secret. When you use the API, you write it yourself.

System prompts can define: a persona, rules to follow, a tone of voice, what topics to avoid, or even a character in a game.

"You are a sarcastic pirate who only speaks in rhyme..." — totally valid system prompt!
Key insight: Every AI chatbot you've ever used had a system prompt. You just couldn't see it.
4
Temperature & Tokens — the knobs
▾
Temperature controls how "creative" or "random" the model is. At 0, it always picks the most likely next word (predictable, focused). At 1, it takes more risks (creative, surprising, sometimes strange).

Tokens are the basic units the model processes — roughly ¾ of a word. max_tokens limits how long the response can be. This also affects cost — you pay per token in production APIs.

Try setting temperature to 0 and asking the same question twice. Then try 1.0 twice. Notice the difference?
Business context: Temperature 0 is best for data extraction. Temperature 0.7–1 is better for marketing copy and creative writing.
5
What is an AI Agent?
▾
An agent is when an AI model's output is used to drive further actions — not just display text to a human. The model becomes a decision-maker in a workflow.

Simple example: User asks "what's the weather in Paris?" → Model says "Paris" → Agent code extracts the city name → Agent calls a weather API → Agent feeds the weather data back to the model → Model gives a full answer.

The model isn't browsing the internet. It's being orchestrated by code that calls tools on its behalf.

In the demo, the "Agent Step" button shows a tiny example: it reads the model's response and decides what kind of follow-up action an agent might take.
The key mental model: Agent = API call + decision logic + more API calls.
